Ontology Specification Draft

Abstract

The DigiChecks Permit Ontology is a top-level ontology for describing permitting processes, requirements, and verification activities. The ontology is based on the Semantic Modelling and Linking (SML, EN 17632) framework and provides core terminology for DigiChecks services to communicate unambiguously about permitting processes.

Permit ontology based on the SML framework: Description back to ToC

The top-level permit ontology contains the relevant terminology related to permitting processes. Based on Semantic Modelling and Linking (SML, EN 17632), it contains the core terminology for DigiChecks services to communicate unambiguously about a permitting process.
